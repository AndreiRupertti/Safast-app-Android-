# Safast-app-Android-
Android application for users in risk situations<br>
An PT-BR application
<br>
<h1>RESUMO (PT-BR)</h1>

O trabalho presente é um aplicativo para Smartphone Android, desenvolvido para auxiliar os usuários antes e depois de situações de risco, tais como incêndios, alagamentos, terremotos e acidentes de trânsito. Através de pesquisas foi relatado o desconhecimento dos cidadãos sobre o que deve, e o que não deve ser feito ao tentar salvar uma vida. O aplicativo tem como objetivo conscientizar e educar os usuários sobre os riscos e as medidas a serem tomadas, ao mesmo tempo que facilita a tomada de decisão rápida e auxílio a vítimas com seu layout limpo e intuitivo, evitando que o nervosismo ou stress atrapalhem-no de chamar as autoridades. Na concepção deste software foram utilizadas as seguintes ferramentas: Java (Android nativo) e XML para o desenvolvimento do aplicativo, para o armazenamento de dados foi utilizado o SQLite e para as criações de ilustrações foi utilizado o Adobe Photoshop CS6. Foram feitos também levantamentos de requisitos, entrevistas e observações pessoais, que possibilitaram ter uma visão do que seria necessário para o desenvolvimento do software. Ele irá possibilitar o usuário a ligar para números emergências rapidamente, ler conteúdos de como agir em situações de risco e terá um alarme sonoro disponível em seu Smartphone.

<br>

<h1>ABSTRACT (EN)</h1>

The present work is an application for Android Smartphone, designed to assist users before and after risky situations such as fires, floods, earthquakes and traffic accidents. Through research it was possible to acknowledge the ignorance of citizens about what they should, and they should not do when trying to save a life. The application aims to educate and consciousness users about the risks and measures to be taken, while facilitating quick decision-making and victim assistance with its clean and intuitive layout, avoiding the nervousness or stress from preventing you to call the authorities. In the making of this software the following tools were used: Java (native Android) and XML for the development of the application. SQLite for the data storage and for the illustrations Adobe Photoshop CS6 was used. There were also surveys of requirements, interviews and personal observations, which made it possible to have a vision of what would be necessary for software development. The application will enable the user to call emergency numbers quickly, read content on how to act in dangerous situations and have a sound alarm available on their Smartphone.
